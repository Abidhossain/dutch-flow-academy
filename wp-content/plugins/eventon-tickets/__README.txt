============================================================
          EventON Addons Installation Guide
============================================================

Welcome to the EventON Addons installation guide! This document will walk you through installing EventON Addons on your WordPress site. EventON Addons extend the functionality of the core EventON plugin, enhancing your event management experience.

============================================================
          Installation Instructions
============================================================

EventON Addons can be installed like any standard WordPress plugin. Follow the steps below:

--- Method 1: Install via WordPress Admin Dashboard ---
1. Log in to WordPress Admin:
   // Navigate to your WordPress admin panel (yoursite.com/wp-admin).
2. Go to Plugins Section:
   // Click on Plugins > Add New in the left-hand menu.
3. Upload the Addon:
   // Click the Upload Plugin button at the top.
   // Select the .zip file of the EventON Addon (do not unzip it).
   // Click Choose File, locate the addon .zip, and select it.
4. Install and Activate:
   // Click Install Now to upload and install.
   // Once installed, click Activate Plugin to enable it.

--- Method 2: Manual Installation via FTP ---
1. Unzip the Addon File:
   // Extract the contents of the EventON Addon .zip file on your computer.
2. Upload to WordPress:
   // Use an FTP client (e.g., FileZilla) to connect to your site.
   // Navigate to the /wp-content/plugins/ directory.
   // Upload the extracted addon folder into this directory.
3. Activate the Addon:
   // Return to your WordPress admin panel.
   // Go to Plugins > Installed Plugins.
   // Locate the EventON Addon and click Activate.

============================================================
          System Requirements
============================================================

To ensure smooth operation, verify your site meets these minimum requirements:
   // WordPress Version: 5.0 or higher
   // EventON Core Plugin: Latest version installed and activated
   // PHP Version: 7.4 or higher
   // Memory Limit: At least 256MB recommended
Check your hosting provider or WordPress admin (Tools > Site Health) for compatibility.

============================================================
          Post-Installation Steps
============================================================

After activating the addon:
1. Verify Activation:
   // Go to EventON > Addons & Licenses to confirm the addon is listed.
2. Enter License Key:
   // Enter and activate the addon to get auto updates and support.
3. Configure Settings:
   // Explore the addon’s settings under the EventON menu to customize.

============================================================
          Updating EventON Addons
============================================================

To download and install updates:
   // Visit: http://www.myeventon.com/documentation/can-download-addon-updates/
   // Ensure your license is active for automatic updates via EventON > Addons & Licenses.

============================================================
          Troubleshooting
============================================================

If you encounter issues:
   // Upload Fails: Ensure the .zip file isn’t corrupted and check server upload limits (php.ini: upload_max_filesize).
   // Activation Errors: Confirm the core EventON plugin is installed and activated.
   // Addon Not Showing: Clear site cache and check file permissions in /wp-content/plugins/ (755 for dirs, 644 for files).
   // Still Stuck? See the Online Documentation or contact support (below).
