Moneybird API koppeling voor WooCommerce
========================================

Integreer Moneybird naadloos in een WooCommerce webshop met deze Moneybird API koppeling.
Genereer en verstuur facturen geheel automatisch of met een simpele druk op de knop.
Deze plugin is compatibel met WooCommerce 2.1.0 en hoger.

Ontwikkelaar: ExtensionTree.com <info@extensiontree.com>

Meer informatie op de product pagina:

https://extensiontree.com/nl/producten/woocommerce-extensies/moneybird-api-koppeling/


Licentie bepalingen
===================

Door het in gebruik nemen van deze plugin gaat u akkoord met de bepalingen beschreven
in het bijgevoegde bestand LICENSE.txt.


Installatie
===========

Upload het ontvangen zip bestand in het WordPress backend, via Plugins -> Nieuwe plugin -> Upload.
Activeer de plugin na de installatie.
De instellingen zijn beschikbaar via WooCommerce Instellingen -> Integratie -> Moneybird API.
Zie de online documentatie voor meer informatie:

https://docs.extensiontree.com/woocommerce-moneybird/
